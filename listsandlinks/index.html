Steps of how to create a new webpage on production server.

  1. After you have made changes to the file (coding html and css) and you must press save.
  2. To see your work on Development Server, you must type python -m SimpleHTTPServer. Make sure that you do this in your REPOSITORY FOLDER.
  3. To validate your code, press control U and copy and paste the code in the html validator for html and css validator for css.
  4. Git add, commit, and push your work in the respository folder in github.
  5. To submit your work to the production server, type sftp username@cmis-hsteacher. If it asks a question, type yes.
  6. Type your password and cd into your public_html. To put the file to the production server, you type put and the file name.
